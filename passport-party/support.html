Support

Need help with Passport Party? This page gives players a simple way to reach support, troubleshoot common issues, and understand what the current launch build can and cannot do.

Contact

Current build

Quick Match is playable today with AI rivals. Private rooms and join codes are a local preview flow in this build and are not yet synced across devices.

Response target

Recommended support promise: reply within 2 business days.

Common questions

  • If progress seems stuck, fully close and reopen the app so local guest progress reloads.
  • If a room code does not behave like live multiplayer, that is expected in the current preview build.
  • If the language looks off in Spanish or Portuguese, the seeded translations may still need polish in this launch version.
  • If you want to erase local progress, use Reset Guest Save inside the app.

What to include in a support email

  • Your device model and iOS version.
  • The room you were playing: Easy, Normal, Hard, or Mixed Global.
  • A short description of what happened.
  • A screenshot if the issue is visual.
